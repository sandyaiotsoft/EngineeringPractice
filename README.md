# EngineeringPractice
信息化项目实战经验沉淀，涵盖产品选型对比、技术方案复盘、踩坑记录与最佳实践。聚焦企业级系统集成、国产化替代、数据平台建设等场景，为同行提供可复用的参考。
## 一、产品简介
金仓数据库（KingbaseES，简称**KES**）是国内领先的**国产数据库**，作为核心**基础软件**产品，专注解决企业级核心系统的**国产化替代**、**信创改造**、**数据库迁移**、**数据库迁移成本**、**数据库运维成本**等关键需求。  

产品具备**高可用**、**高性能**、**全兼容**、**易运维**四大核心特性，全面适配**鲲鹏**、**飞腾**等国产芯片，以及**欧拉**、**麒麟**等国产操作系统，支持**KSql**语法，可提供**数据库一体机**、**快速开发平台**、**数据同步软件**、**数据集成平台**等配套方案，已在金融、政府、能源、医疗、交通等关键行业落地，为用户提供安全可靠的数据存储与管理解决方案。  

本仓库为金仓数据库产品手册库，同时提供**金仓社区**入口，包含从部署到运维的全套技术资料，适用于开发工程师、运维人员、架构师在**数据库选型**、**系统集成**、**迁移实施**、**信创数据库选型**、**国产数据库选型**、**国产数据库选型推荐**等场景下快速参考。

### 2. 政府行业：某省级政务云信创改造
